# Lu

>这是一个多屏互动的美女脱衣服的小游戏，美女的设计原型来自我们的交互设计师**小路**，游戏本身也取意“撸一撸”，所以最终项目取名 `Lu`。L大写，u小写。

## 玩法

1. 安装 [Node.js]()
2. `npm install node-dev -g`
3. `npm install && npm start`
4. 在浏览器中打开 [http://localhost:3000/screen]
5. 确保手机和打开上面网址的电脑在同一局域网
6. 在手机上打开任意扫码工具扫描
7. 喊其他小伙伴一起来扫码“撸一撸”
8. 所有人都加入后，点击电脑上的开始按钮

Enjoy！
